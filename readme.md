# Backend Gerenciamento de livros didáticos na biblioteca

## Instalação
1. Clone este repositório:
```
git clone https://github.com/T0ntow/back-library-FMD.git
```
2. Navegue até o diretório do projeto:
cd back-library-FMD

3. Instale as dependências:
```
npm install
```

## Executando o servidor 
Para iniciar o servidor, execute o seguinte comando:
```
node server.js
```
